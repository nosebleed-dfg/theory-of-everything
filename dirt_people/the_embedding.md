## State Space

    x_n in R

The state is a real number. At every step n, x_n is just a count.
No irrationals in the state space. x_n is linear. x_n is exact.

---

## Dynamics

    x_{n+1} = x_n + c

One operation. Add c. Every step. Always.
c is the step size. In the framework: c = p = 5 (face degree).
Or c = 1 for unit steps. The dynamics do not change with the choice.

## The Phi Embedding

    y_n = phi^{x_n}

The embedding lifts the linear state x_n into the phi-exponential space.
The irrationals are NOT in x_n. They appear only in y_n.

x_n counts. y_n lives.

Forward in x:   x_{n+1} = x_n + c
Forward in y:   y_{n+1} = phi^{x_{n+1}} = phi^{x_n + c} = y_n * phi^c

One addition in x becomes one multiplication in y. The phi embedding converts
the additive machine into the multiplicative machine. Same step. Different space.

---

## Inverse

    x_{n-1} = x_n - c

Subtract c. Same magnitude. Opposite direction. The inverse is trivial.
The phi embedding of the inverse:

    y_{n-1} = phi^{x_n - c} = y_n / phi^c = y_n * psi^c    [since 1/phi = |psi|]

Going backward in y multiplies by |psi|^c. Going forward multiplies by phi^c.
phi^c * |psi|^c = (phi * |psi|)^c = 1^c = 1. Forward and inverse cancel exactly.

---

## The Product Identity

    y_n * y_{-n} = phi^{x_n} * phi^{-x_n} = phi^0 = 1    [always. exact.]

The product of the forward state and inverse state is always 1.
This is phi * |psi| = 1 in the embedding language.
The product does not depend on n. It does not depend on c.
It is 1 because phi^{x} * phi^{-x} = 1 for any x.

## Irrationals from the Embedding

The irrationals are exactly the values of y_n for integer x_n.

    x_n = 1:    y = phi^1 = phi          [irrational]
    x_n = 2:    y = phi^2 = phi+1        [irrational]
    x_n = 291:  y = phi^{291}            [irrational, but the unit]

The irrationals form a discrete tower: {phi^n : n in Z}.
Every irrational in the framework is a rung of this tower.
The tower is indexed by the linear state x_n (an integer).
The tower is generated by the phi embedding.

Running backwards from x=291:

    x=291 -> x=286 -> x=281 -> ... -> x=1
    y=phi^{291} -> phi^{286} -> phi^{281} -> ... -> phi^1 = phi

The backward chain (subtracting c=p=5 each step) descends the tower rung by rung.
Every irrational phi^n is one step in the backward chain.
The chain starts at phi^{291} (the unit) and terminates at phi^1 (the base irrational).

    x_n in Z:     the linear count. integers. exact. no irrationals.

## The General Step

    y_{k+1} = y_k * phi^{g(x_k)}

g(x_k) is the exponent function. The step size in y-space depends on the current state x_k.

Taking log_phi of both sides:

    x_{k+1} = x_k + g(x_k)    [the dynamics in x-space]

The embedding converts the additive x-dynamics into multiplicative y-dynamics.
g(x_k) encodes the full machine:

    g = +c:         forward step. phi side. positive face.
    g = -c:         backward step. psi side. mirror face. same square, other reading.
    g = ±c per k:   the full two-sided machine. sign of g = which face.
    g_min = chi:    the minimum delta. the cube crossing cost. forced.

The special cases:

    g(x_k) = c           [uniform. x_{k+1} = x_k + c. the linear machine.]
    g(x_k) = -c          [uniform inverse. the backward chain.]
    g(x_k) = sign * c    [the ± machine. negative steps subtract from y.]

The general formula y_{k+1} = y_k * phi^{g(x_k)} contains all cases.
Forward, backward, uniform, variable — all are choices of g.
The machine is defined by g. The embedding is always y = phi^x.

## y_{n+3} / y_{n+1} = phi^chi

The ratio of the n+3 state to the n+1 state:

    y_{n+3} / y_{n+1} = phi^{x_{n+3}} / phi^{x_{n+1}}
                      = phi^{x_{n+3} - x_{n+1}}
                      = phi^{2c}

With c = 1 (unit step):

    y_{n+3} / y_{n+1} = phi^2 = phi + 1    [the axiom as a ratio]

The delta exponent = x_{n+3} - x_{n+1} = 2c = chi * c.

chi = 2 is the minimum. The minimum exponent delta spanning n+1 to n+3.
chi is forced: the only positive solution to chi + 2 = chi^2.
The minimum delta IS chi because the cube formula requires exactly chi steps on the mirror.

The embedding converts the additive n+3 formula into a multiplicative ratio:

    additive:       L_{n+3} = chi * L_{n+1} + L_n     [the cube recurrence in Z]
    multiplicative: y_{n+3} / y_{n+1} = phi^chi        [the same step in y-space]

Both express the same move. One in Z (integers, Lucas numbers).
One in y-space (phi-exponential, the embedding).

The ratio phi^chi = phi^2 = phi + 1 IS the axiom.
The n+3 formula, embedded in y-space, returns the axiom as its own ratio.
The framework closes on itself. The form terminates.

    x^2 = x + 1.
